# RIPPLE Onboarding & Placement by VITRA

## Overview

RIPPLE Onboarding & Placement by VITRA is a Flask portfolio prototype for a premium employee onboarding and placement intelligence experience.

The product guides an employee through a 10-question open-response onboarding sequence, creates a soft employee-facing profile, and reuses the same demo scoring data for an HR placement report.

## Product Direction

RIPPLE is designed around a white-water, glasslike, futuristic interface:

- white and soft silver backgrounds
- CSS ripple and water-caustic texture
- luminous frosted panels
- circular geometry
- quiet black typography
- calm cinematic spacing
- responsive full-screen onboarding and HR views

## Current Features

- Splash / entry page with RIPPLE identity
- Demo access page with Company / HR and Employee entry paths
- Employee welcome page
- Reusable 10-division question sequence
- Processing state with glowing ripple visual
- Employee Results page
- Placement Path page with role alignment percentages
- HR Dashboard
- HR Placement Report
- Five archetypes:
  - Architect
  - Current
  - Signal
  - Spark
  - Anchor

## Routes

- `/` - Splash / Entry
- `/access` - Demo access and login-style entry
- `/welcome` - Employee welcome
- `/questions` - 10-question sequence
- `/processing` - Processing transition
- `/results` - Employee Results
- `/placement` - Placement Path
- `/hr` - HR Dashboard
- `/hr/candidate/<candidate_id>` - HR Placement Report

Legacy routes `/assessment`, `/pathways`, and `/about` redirect into the new RIPPLE flow.

## Scoring Model

The prototype uses simple demo scoring:

- open-response keyword matching
- optional signal chips on each question
- five archetype score buckets
- one shared score result for employee and HR views

This is not real AI, not a validated psychometric tool, and not a hiring decision system.

## Prototype Honesty

This repository is a portfolio prototype.

- Authentication is demo-only.
- Employee responses are stored in local browser storage.
- There is no production database.
- There is no real encryption or secure account system.
- The HR dashboard uses demo candidate data.
- Placement recommendations are generated by simple keyword and chip scoring.

## How To Run

1. Create or activate your virtual environment.
2. Install dependencies:
   - `pip install -r requirements.txt`
3. Start the application:
   - `python .\app.py`
4. Open:
   - `http://127.0.0.1:5000`

## Full Demo Journey

1. Open `/`.
2. Choose Enter RIPPLE.
3. Use either Employee Access or Company / HR Login.
4. Continue to the employee welcome page.
5. Complete all 10 question divisions.
6. Wait through the processing page.
7. Review the Employee Results page.
8. Open the Placement Path page.
9. Open `/hr` for the HR Dashboard.
10. Select a candidate to view the HR Placement Report.
